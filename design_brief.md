# CM Predictor — Design Brief

## Что за приложение

Telegram Mini App для прогнозов на футбольные матчи. Пользователи выбирают чемпионат и дату, видят расписание матчей, делают прогноз на счёт (из счёта автоматически выводится исход: победа хозяев / ничья / победа гостей), зарабатывают очки и соревнуются в таблице лидеров.

## Экраны

### 1. Авторизация
- Два таба: **Войти** / **Регистрация**
- Поля: логин, пароль
- Кнопка действия
- Логотип приложения сверху (⚽ или футбольная тема)

### 2. Главная — список матчей
Верхняя часть (фильтры):
- Дропдаун выбора чемпионата (Premier League, UCL, Serie A и др.)
- Выбор даты (инпут типа date)
- Кнопки быстрой навигации: ← вчера / сегодня / завтра →

Список карточек матчей.

### 3. Карточка матча
```
┌─────────────────────────────────────┐
│ BCN 21:00  ZRH 21:00  UFA 02:00  [Статус] │
├─────────────────────────────────────┤
│  [Герб]         vs        [Герб]   │
│ Manchester City      Arsenal        │
├─────────────────────────────────────┤
│  Прогноз на счёт:                   │
│                                     │
│     [+]           [+]               │
│      2      :      1                │
│     [−]           [−]               │
│                                     │
│  [Победа хозяев]    [Сохранить]     │
└─────────────────────────────────────┘
```

Состояния карточки:
- **Запланирован** — показывается форма прогноза со степперами
- **Идёт сейчас** — текущий счёт, живая индикация
- **Завершён** — итоговый счёт + показывается сделанный прогноз и заработанные очки

### 4. Таблица лидеров
- Строка текущего пользователя выделена
- Колонки: место, имя, очки
- Фильтр по чемпионату
- Медали для топ-3 (🥇🥈🥉)

### 5. Нижняя навигация (Tab Bar)
Три таба: ⚽ Матчи / 🏆 Лидеры / 👤 Профиль

## Система очков

- Правильный исход (победа/ничья): **1 очко**
- Точный счёт: **+3 очка** (дополнительно к очку за исход)

---

## Ключевые UX-детали

- Время матча отображается сразу в трёх часовых поясах: **BCN / ZRH / UFA**
- Прогноз вводится через степперы `+` / `−` (не текстом) — исход вычисляется автоматически
- Отрицательный счёт невозможен
- Мобильный формат, max-width 480px
- Тёмная тема (основная и единственная)

---

## Платформа

Telegram Mini App, открывается внутри Telegram. Интерфейс должен быть адаптирован под мобильные экраны, тач-управление, без hover-эффектов как основных.
